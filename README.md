# 📁 Folder Architect

## 📌 Project Description

Folder Architect is a web-based application that simulates a real-world file system using a folder-based architecture. It allows users to create, manage, and retrieve data through a hierarchical structure of folders and files.

---

## 🎯 Objective

The objective of this project is to understand how data can be structured, stored, and retrieved efficiently using a tree-based folder system, similar to modern operating systems.

---

## 🚀 Features

### ✅ Core Features

* Create folders (including nested folders)
* Add files inside folders
* Navigate between folders
* View folder contents (files and subfolders)

### 🌟 Advanced Features

* 🔍 Search functionality for files and folders
* 🗑️ Delete files and folders
* ✏️ Rename files and folders
* 📂 Breadcrumb navigation (Root > Folder > Subfolder)
* 📊 Display file and folder count
* 💾 Persistent storage using localStorage
* 🎨 Dark mode UI

---

## 🧠 Concepts Used

* Tree Data Structure (for folder hierarchy)
* CRUD Operations (Create, Read, Update, Delete)
* DOM Manipulation (JavaScript)
* Event Handling
* Local Storage for data persistence

---

## 🛠️ Technologies Used

* HTML
* CSS

---

## ⚙️ How It Works

* The application starts with a root folder.
* Users can create folders and files inside it.
* Each folder can contain subfolders and files.
* The structure is dynamically updated and displayed.
* All data is stored in the browser using localStorage.
* Users can search, rename, delete, and navigate through folders.
